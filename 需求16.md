现在只集成 wrist_rotation.py，不要改现有 pinch/haptic/1-back 行为。

当前 wrist_rotation.py 和 tests/test_wrist_rotation.py 已经存在，但 run_pinch_haptic_1back.py、dualtask_config.yaml、dualtask_logger.py 还没有实际集成 wrist_rotation。

目标：
把 wrist_rotation 作为可选记录模块接入正式双任务 runner。
默认 enabled=false，不影响现有实验。

修改要求：

1. dualtask_config.yaml 增加：

wrist_rotation:
  enabled: false
  node_id: 0
  quaternion_order: wxyz
  calibration_duration_s: 3.0
  min_valid_frames: 30
  feature_method: calibrated_axis_projection
  classification_margin: 0.15
  save_timeseries: true

2. run_pinch_haptic_1back.py：
如果 wrist_rotation.enabled=false，现有流程完全不变。

如果 wrist_rotation.enabled=true：
- 在 pinch calibration 之前或之后，增加三段 wrist calibration：
  neutral wrist
  left wrist rotation
  right wrist rotation
- 每段采集 calibration_duration_s 的 node0 rotation quaternion。
- 调用 wrist_rotation.calibrate_wrist_rotation。
- 保存 wrist_rotation_calibration.json。
- 如果 calibration failed，summary 写 warning，但不要让整个实验崩溃；除非 config 以后增加 required=true。
- 正式阶段每次有新 MANUS frame 时，调用 classify_wrist_rotation_frame。
- 写 wrist_rotation_timeseries.csv。

3. dualtask_logger.py：
增加：
- wrist_rotation_calibration_json
- wrist_rotation_timeseries_csv
- write_wrist_rotation_calibration()
- write_wrist_rotation_sample()

wrist_rotation_timeseries.csv 字段至少包括：
session_id
wall_time_iso
monotonic_ms
source_frame_id
node_id
q_w
q_x
q_y
q_z
wrist_rotation_valid
wrist_rotation_score
wrist_rotation_class
distance_to_left
distance_to_right
note

summary.json 增加：
wrist_rotation_enabled
wrist_rotation_calibration_passed
wrist_rotation_failure_reason
wrist_rotation_timeseries_csv

4. 测试：
新增或补充：
- enabled=false 时不创建 wrist files，不影响现有 runner core。
- enabled=true 时 fake MANUS frames 可以完成 neutral/left/right calibration。
- enabled=true 时正式阶段会写 wrist_rotation_timeseries.csv。
- calibration failed 时 summary 有 warning，但主流程不崩。
- 缺少 node0 rotation 时写 invalid sample，不崩。

5. 不要修改：
- exp2
- manus_vive_com
- MANUS TCP 8888
- haptic scheduler
- SimpleHapticSender
- nback_timeline
- pinch calibration

6. 运行：
python -m py_compile wrist_rotation.py run_pinch_haptic_1back.py dualtask_logger.py
python -m pytest tests/test_wrist_rotation.py tests/test_pinch_haptic_1back_core.py
